# bootcamp_java_springboot
Construa uma API com Java e Spring de ponta a ponta!  Vá do básico de uma das principais linguagens back-end até práticas avançadas com Spring, aplicando metodologia de testagem TDD e criando uma visão completa de system design com arquitetura de microsserviços, e faça deploy em nuvem com o AWS e copilotos de IA.
